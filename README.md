**Caesar Cipher in Python**

A lightweight, interactive command-line tool implemented in Python that encrypts and decrypts text using the classic Caesar Cipher algorithm.

 **Overview:-**

The Caesar Cipher is one of the simplest and most widely known encryption techniques. It is a type of substitution cipher in which each letter in the plaintext is replaced by a letter some fixed number of positions down the alphabet.

This program allows users to easily encrypt secret messages or decrypt existing ciphertexts by specifying a custom shift value.

**Features**
Two-Way Operation: Supports both encryption and decryption from a single intuitive menu.

Case Preservation: Accurately maintains the uppercase and lowercase formatting of the original message.

Character Filtering: Safely ignores spaces, numbers, and special punctuation, leaving them unaltered in the final output.

Modular Arithmetic: Handles shift wrapping seamlessly (e.g., shifting 'Z' by 1 wraps around to 'A').

Input Validation: Prevents crashes by handling invalid mathematical inputs for the shift value.
